# API Endpoints

## HTML API

### Root

* GET / - loads React web app

## JSON API

### Users

* POST /api/users
* GET /api/users/:id

### Playlists

* GET /api/users/playlists
* GET /api/users/playlists/:id
* POST /api/users/playlists
* PATCH /api/users/playlists/:id

### Session

* POST /api/session
* DELETE /api/session

### Artists

* GET /api/artists
* GET /api/artists/:id

### Songs

* GET /api/songs
* GET /api/songs/:id
